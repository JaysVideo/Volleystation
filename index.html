⚡ Proxy Setup (one-time, 30 seconds)

Browsers block direct API calls to external servers (CORS policy). A tiny local proxy script forwards your requests. It runs on your machine — your token never leaves your network.

1. Make sure Node.js is installed (nodejs.org)

2. Save proxy.js from this repo to any folder and run:

node proxy.js

3. Leave that terminal open while using the dashboard. The proxy runs at http://localhost:3030.

That's it — no npm install needed. The proxy is a single file, ~40 lines.
